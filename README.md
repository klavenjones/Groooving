## Objective

A Playlist Manager for Spotify, built with React.js, that will allow Spotify account holders to search for songs from Spotify and build a playlist that they would like to save to their existing account. 

## Technologies used

- HTML5 
- CSS
- Node.js
- React.js
- Spotify API
